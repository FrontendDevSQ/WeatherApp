# Weather App

Weather App is a web application that provides real-time weather information for any location worldwide.  
It displays temperature, humidity, wind speed, and weather conditions, and includes a clean interface, interactive map, favorites/bookmarks, weather alerts, dark mode toggle, and current location support.

## Features

- Real-time weather information (temperature, condition, humidity, wind speed)  
- Current location weather  
- Favorites / Bookmarks for saved locations  
- Dark mode toggle  
- Interactive map with location marker  
- Search by city name  

## Tech Stack

- HTML5  
- CSS3 (Custom + Flex/Grid)  
- JavaScript (Vanilla JS)  
- [OpenWeatherMap API](https://openweathermap.org/api)  
- [Leaflet JS](https://leafletjs.com/) for maps  
- [Font Awesome](https://fontawesome.com/) for icons
